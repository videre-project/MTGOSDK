# API Documentation

Welcome to the **MTGOSDK** API documentation. This reference provides detailed information about the classes, methods, and properties available in the SDK.

## Key Namespaces

### [MTGOSDK](xref:MTGOSDK)
The core namespace containing the main entry points for interacting with the MTGO client, including `Client`, `Events`, and game-related classes.

### [MTGOSDK.MsBuild](xref:MTGOSDK.MSBuild)
Contains build-time tools and source generators used by the SDK.

### [MTGOSDK.Win32](xref:MTGOSDK.Win32)
Provides low-level Win32 API wrappers and utilities.
